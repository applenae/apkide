# APK-IDE

APK-IDE是一款用于反编译，编译和修改APK的安卓软件。
内部使用 ApkTool 2.7.0

## 功能

- [x] 访问存档文件中的 class 文件
- [x] 反编译和编译 apk 文件
- [x] aapt (专门为apktool使用. v1/v2)
- [ ] apk 签名
- [ ] apk 对齐优化
- [ ] git 集成
- [ ] 反编译与构建 日志输出
- [ ] 可配置的 ApkTool(Framework...) 选项
- [ ] 快速编辑大文本文件
- [ ] 语法高亮 (smali, java, c/c++, xml)
- [ ] 代码补全 (仅 smali?)
- [ ] 代码分析与重构 (仅 smali?)
